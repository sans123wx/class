过度

一、transition-delay
  1. 作用：设置延迟时间
  2. 值：<时间>
  3. 示例代码： 
            .delay:hover {
            color:red;
            background-color:green;
            transition-delay:100ms;
            -webkit-transition-delay:100ms
        }

  4. 示例：

    鼠标放上去以后，延迟为5秒
二、transition-duration
  1. 作用：过渡时间
  2. 值：<时间>
  3. 示例代码： 
            .duration:hover {
            color:red;
            background-color:green;
            transition-duration:100ms;
            -webkit-transition-duration:100ms
        }

  4. 示例：

    鼠标放上去以后，过渡时间设置为2秒
